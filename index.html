UNSERE IDEE

Kernidee unseres Startups

Erster Schritt

Der Kunde kann es sich auf einem FlyingFood Sitzbank gemütlich machen. Mithilfe einer Web App kann man ein Menü zusammenstellen und bestellen.

Zweiter Schritt

Das Menü wird von einem lokalen Restaurant hergestellt und anschließend in einer Aluminium Box an die Drohne angebracht.

Dritter Schritt

Die Drohne liefert das Menü innerhalb weniger Minuten. Die Box wird dabei bei der Landung auf dem Landeplatz deponiert.

Vierter Schritt

Die Aluminium Box kann nach dem Essen wieder auf der Landeplattform deponiert werden. Anschließend wird die Box mittels Knopfdruck auf der Webapp von der Drohne wieder abgeholt.

Wir wissen wie träumen geht.
