UNSERE IDEE

Kernidee unseres Startups

Erster Schritt

Der Kunde kann es sich auf einem FlyingFood Sitzbank gemütlich machen. Mithilfe einer Web App kann man ein Menü zusammenstellen und bestellen.

Zweiter Schritt

Das Menü wird von einem lokalen Restaurant hergestellt und anschliessend in einer Aluminium Box an die Drohne angebracht.

Dritter Schritt

Die Drohne liefert das Menü inerhalb weniger Minuten. Die Box wird dabei bei der Landung auf dem Landeplatz deponiert.

Vierter Schritt

Die Aluminium Box kann nach dem Essen wieder auf der Landeplattform deponiert werden. Anschliessend wird die Box mittels Knopfdruck auf der Webapp von der Drohne wieder abgeholt.

Unsere Geschichte

  • Aug. 2022
    Erste Vorstellung
    In einem Café in Thun wird die Idee erstmals vorgestellt. Alle Anwesenden sehen das Potenzial der Idee.
  • Dez. 2022
    Businessplan & Finanzierungsplan
    Der Business- und Finanzierungsplan wurde erstellt. Diese zwei Konzeptpläne beweisen wie rentabel das System ist.
  • März 2023
    1. Prototype
    Der erste Protorype einer Autonomen Drohne schliesst erfolgreich ihren Testflug ab.
  • Apr. 2023
    Absturz
    Aufgrund einer zu wenig gut befestigten Luftschraube welche im Flug abfliegt, kommt der 1. Prototype in schieflage und stürzt von einer Höhe von 48 Meter zu Boden.
  • Mai 2023
    Drohne V3
    Die 3. Version der Drohne wurde konstruiert.
  • 2023- Heute
    Bau Drohne V3
    Die Drohne V3 ist im Bau. Sie wird eine höchstgeschwindigkeit von 90 Kmh haben, dabei 20 Minuten in der Luft sein können und eine Maximale Nutzlast von 9.5 Kilogramm haben. Dies alles bei eine Thrust Weight Faktor von mehr als 2:1.
